# route.lw1.at

All files and code powering [route.lw1.at](https://route.lw1.at) are in this repository.

- See https://about.route.lw1.at for more information
- See [git.lw1.at/lukas/motis-fork](https://git.lw1.at/lukas/motis-fork/compare/master..route.lw1.at-fork) for the patches applied on top of MOTIS
- [config.yml](./config.yml) is the config file I use for MOTIS
- The files in [datasets/](./datasets/) are using [git-annex](https://git-annex.branchable.com/). I am thinking about some ideas how I can keep historical versions of GTFS Feeds publicly available.
